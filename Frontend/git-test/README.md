## Git Command

- `git init`
  git 감지할수 있는 .git 파일 생성
- `git remote add origin <remote repository url>`
  push 할 경로 설정
- `git add <file name>`
  git 스테이징 상태로 변경
- `git commit`
  git 커밋메시지 작성
- `git push origin <branch name>`
  git 클라우드로 이동
- `git pull origin <branch name>`
  git 변경된 사항 다운
- `git merge <branch name>`
  브렌치 메인에 합병
